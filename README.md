## Speciális karakterek
### Windows
| Karakter | kód              | Karakter | kód            |
|----------|------------------|----------|----------------|
| [        | AltGr + F        | ]        | AltGr + G      |
| {        | AltGr + B        | }        | AltGr + N      |
| <        | AltGr + í vagy m | >        | AltGr + y      |
| &        | AltGr + C        | \|       | AltGr + w      |
| *        | AltGr + -        | ;        | AltGr + ,      |

### OS X
| Karakter | kód             | Karakter | kód             |
|----------|-----------------|----------|-----------------|
| [        | alt + 8         | ]        | alt + 9         |
| {        | alt + 7         | }        | alt + ö         |
| <        | alt + shift + y | >        | alt + shift + x |
| &        | alt + 1         | \|       | alt + í         |
| *        | alt + shift + , | ;        | alt + .         |
